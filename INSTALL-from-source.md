Nanodash Installation from Sources
==================================

First clone the repository:

```bash
git clone https://github.com/knowledgepixels/nanodash.git
cd nanodash
```

### Run for development

Build and run Nanodash on http://localhost:37373

```bash
./mvnw clean jetty:run
```
